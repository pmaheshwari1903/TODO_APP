# 📝 Todo App

A simple and beginner-friendly **Todo List Web App** built using **HTML, CSS, and JavaScript**.
This project allows users to add tasks, prevent duplicate entries, and remove tasks easily.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Remove tasks
* ⛔ Prevent duplicate tasks
* ⌨️ Add task using **Enter key**
* 🎯 Input validation (empty tasks not allowed)
* 🎨 Clean dark-themed UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

A simple UI where users can:

---

## ⚙️ How It Works

### 1. Adding Tasks

* User enters text in input field
* Clicks **Add** or presses **Enter**
* Task is added to the list

### 2. Validation

* Empty input → alert shown
* Duplicate task → alert shown

### 3. Removing Tasks

* Each task has a **Remove** button
* Clicking it deletes the task instantly

---

## 💡 Key Logic Used

* `trim()` → removes extra spaces
* `querySelectorAll()` → checks duplicate tasks
* `addEventListener()` → handles button & keyboard events
* DOM Manipulation → creates and removes elements dynamically
